== Descripció del projecte

EasyTraza és una aplicació client-servidor orientada a la gestió de traçabilitat dins d’un entorn de producció alimentària.

El projecte està format per:

* un *backend* desenvolupat amb *Spring Boot*
* un *client web* desenvolupat amb *HTML, CSS i Thymeleaf*
* un *client mobile* desenvolupat amb *Kotlin i Jetpack Compose*

== Funcionalitats implementades

=== Sprint 1

* RF03: CRUD d'usuaris
* RF04: CRUD de matèries primeres
* RF08: CRUD de proveïdors
* RF15: Configuració persistent de la IP i connexió entre mobile i backend
